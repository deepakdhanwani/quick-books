# Quick Books — Admin Web

Admin portal built with **Expo SDK 54** (React Native Web).

**Port:** `9091`  
**Backend API:** `http://localhost:9090`

## Setup

```bash
cp .env.example .env
npm install
```

## Run

```bash
npm run web
```

Opens at http://localhost:9091

Login with `ADMIN_EMAIL` / `ADMIN_PASSWORD` from `backend/.env`.
